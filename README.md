# ☕ Coffee Heaven

A modern and aesthetic coffee website built using HTML and CSS.  
This project showcases a complete coffee brand experience — from menu to the journey of coffee beans.

---

## 🚀 Features

- 🎨 Beautiful hero section with overlay
- 📋 Interactive menu cards
- 🌟 Specialities section with animations
- 🌱 Coffee journey (Plantation → Brewing)
- 📱 Responsive design
- 🎯 Clean UI with hover effects

---

## 🛠️ Built With

- HTML5  
- CSS3  
- Flexbox  
- Animations
